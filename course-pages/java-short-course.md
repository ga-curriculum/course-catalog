| [Java Objects and Classes](https://github.com/ga-curriculum/java-objects-and-classes) | Lecture | 60 min | Learn to instantiate objects, add fields and methods to classes, and write constructors. |
| [Working with Files in Java Lab](https://github.com/ga-curriculum/working-with-files-in-java-lab) | Lab | 45 min | Build a Java application that performs file input and output operations. |
| [Java Subclasses](https://github.com/ga-curriculum/java-subclasses) | Lecture | 75 min | Explore inheritance using `extends`, `@Override`, and subclass/superclass relationships. |

| [Java Inheritance and Abstraction Lab](https://github.com/ga-curriculum/java-inheritance-and-abstraction-lab) | Lab | 75 min | Practice modeling a vehicle hierarchy using abstract classes and interfaces. |
